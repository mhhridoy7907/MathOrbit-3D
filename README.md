#  MathOrbit-3D

An interactive **3D Math Graph Visualizer** built using **Three.js**, allowing real-time manipulation of Sin and Cos wave functions in a 3D space.

---

##  Features

-  Real-time Sin & Cos graph rendering
-  Custom input controls (A, B values)
-  Adjustable X, Y, Z offset positioning
-  3D Orbit controls (rotate, zoom, pan)
-  Grid & Axis helper for better visualization
-  Fixed 3D Text label ("MH HRIDOY")
-  Lightweight and browser-based (no install needed)
